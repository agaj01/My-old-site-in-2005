4.      简述太阳系的结构、形成和演化。

5.      地球有哪些圈层结构？它们是如何形成的？

 

第三节  生命的起源

简单叙述生命的起源可分为哪些层次和阶段？

 

第三章        现代高新技术

 

第一节  生物技术

        6．什么是纳米材料？它的特点和用途是什么？

第五节  通信技术

第四章   环境保护和可持续发展

 

第一节  环境与环境问题

1.    什么叫环境？ 按照人类的生存环境，它可以分为哪几类？

2.    什么叫生态系统？生态系统具有哪些功能？

3.    举例说明什么叫生态平衡和生态破坏？

4.    举例说明什么是第一环境问题、第二环境问题、第三环境问题？

5.    环境问题按环境要素、污染物的性质和污染物的形态各分为几类？

6.    试述当前环境问题的主要特点。

 

第二节  全球性环境问题

    1.    什么叫温室效应？使温室效应加剧的温室气体有哪些？简述气候变暖的原因及危害。

    3.    生物多样性的涵义是什么？

    4.    什么是赤潮？试述赤潮形成的原因。

 

第三节  环境污染

    1.    表示水体有机污染的主要指标是什么？

    2.    煤烟型污染和光化学烟雾污染中的主要污染物各是什么？简述污染物的来源。

    3.    简述酸雨的形成及其危害。

    4.    你认为我国当前主要的环境问题是什么？应当采取哪些措施或加强哪方面的工作？

 

第四节  生态破坏
